Detailed Description

Local Variable.

Local variables are named variables defined in the source-code. Each of them carries a name and a type associated with them along with the source reference where it was defined in the code.

Source reference where local was declared.

The source reference is returned as an immutable reference. It is owned by the CompilerContext which outlives the MIR, so it is guaranteed to live at least as long as the MIR.
