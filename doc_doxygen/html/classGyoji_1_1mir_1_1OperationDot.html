Detailed Description

This subclass of Operation is used to access member variables of classes and other aggregate types by reference.

This class represents accessing a class member using the name of the member and the operand which is a reference to the class itself. The returned operation will be an lvalue if the member variable is an lvalue (which is almost certainly will be).
