Detailed Description

Represents a pointer to a function.

This type represents a pointer to a function. This is designed to follow the c-style syntax of function pointers. This isn't the most efficient or readable way to specify function pointers, but is is traditional and well-known to those familiar with the C syntax.

This represents a function pointer type as expressed by the typical C syntax for function pointers:

    type_specifier ( * identifier_token )(function_definition_arg_list)

This specifies the name given by "identifier_token" as a pointer to a function returning the type_specifiere and receiving the arguments given.
